ANALISIS Y PREDICCION DE MEMES :'v

Este programa es el proyecto final para la materia de Inteligencia Artificial el objetivo de este programa es indicar al usuario si la imagen que selecciono es un Meme o no.

Para desarrollar este programa se utilizaron los siguientes programas:

  • OpenCV 3.1.0
  • Boost
  • NetBeans 8.2
  • BulkImageDownloader
  • RenameElite
